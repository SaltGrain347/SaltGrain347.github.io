Baldi's Basics Mods Salt's Pick

since the Baldi's Basics Classic released, the Gamebanana page is filled with shovelware/horrifying-badly made mods ever. I've basically dedicated to picking the god honest highest quality effort of bunch of mods!
also if you want me to list some mods that I've missed, contact through my socials

(Note: this list will not contain any regular reskins at all. the mod must atleast contain custom contents than having of added BB+ contents.)

Raldi's Crackhouse

Raldi's Crackhouse is a chaotic meme-heavy mod with absurd humor while having multiple game modes, and an unique boss fight involving Atlas Earth.

Dave's Fun Algebra Class: Remastered

Dave's Fun Algebra Class: Remastered is a Remaster of Dave's Fun Algebra Class which basically has entirely different map, character and items. It's also the only Baldi Mod/Fangame to be on Steam.

Yin's Birthday Bash Reimagined

Yin's Birthday Bash Reimagined is a Reimagined/Remaster of a Yin's Birthday Bash where you enters Yin's Birthday Party, has an unique gameplay gimmick and a Unique Phase 2 Boss fight.

Foxo's Fun Schoolhouse

Foxo's Fun Schoolhouse is an unique fangame where it has multiple levels that can be accessed via elevator from first level, Two Possible Ending Path, and a lots of secrets. The style is really leans on early educational computer game.
Also has a upcoming version 2.0, this link leads to a demo at the moment.

Jack's Basic Algebra Fundamentals

Jack's Basic Algebra Fundamentals is a (currently a demo) mod of Baldi's Basics Classic where it contains a score system, hard math equation, unique items, lapping system and more modes soon to comes!

Thinker's Robotic House

Big Thinker's Robotic House is an absurd yet really unique baldi fangame yet to be released, contain really unique Items to defend yourselves from Thinker and collect Pez with 3 Activities.
